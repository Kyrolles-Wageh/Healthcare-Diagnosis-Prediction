# 🏥 Healthcare Diagnosis Prediction

This project focuses on predicting medical test results using Machine Learning techniques.

## 📊 Project Overview
The goal of this project is to analyze healthcare data and build predictive models for medical test outcomes (Normal, Abnormal, Inconclusive).

## ⚠️ Important Insight
Initial results showed low accuracy (~35%), which revealed that:
- The available features are not strongly correlated with test results
- The dataset lacks critical medical indicators (e.g., lab values, vital signs)

## 🧠 Models Used
- Logistic Regression
- Decision Tree
- Random Forest

## 📈 Evaluation Metrics
- Accuracy
- Precision
- Recall
- F1-score

## 🔍 Key Findings
- Balanced dataset (~33% per class)
- Weak correlation between features and target
- Random Forest performed better than other models

## 📊 Visualizations

- Feature Importance
